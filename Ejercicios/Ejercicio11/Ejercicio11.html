Lenguaje de Programación JavaScript

Conceptos básicos

JavaScript es un lenguaje de programación interpretado, orientado a objetos, débilmente tipado y dinámico.

JavaScript se utiliza principalmente en la creación de páginas web dinámicas, en el cual se pueden realizar animaciones, efectos especiales, y acceso a elementos de la página.

Cada variable en JavaScript debe tener un nombre único. El nombre de la variable puede contener letras, dígitos o los símbolos $ y _.

JavaScript tiene una gran variedad de tipos de datos, como números, cadenas de texto, booleanos, objetos, entre otros.
